## schema's

Selecteer de app (Open) Register en maak de volgende schema's aan via **Schemas** -> **Add schema**:
- Publicatie
- Organisatie
- Bijlagen
- Catalogus
- Directory
- Thema
- Publicatie Type
- Pagina

![Confugureren schema's](../../../static/img/OpenCatalogi/OC_schemas.png)

Het eindresultaat ziet er dan zo uit.
