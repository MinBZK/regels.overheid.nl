 # Inleiding

 OpenFisca is een open-source softwareframework voor het modelleren van belasting- en uitkeringsstelsels op een manier die zowel leesbaar is voor mensen als uitvoerbaar is voor computers. Het maakt het mogelijk om wet- en regelgeving te vertalen naar uitvoerbare Python-code, waardoor deze toegankelijk wordt voor zowel juridische als technische experts.
